## Getting Started

First, if database data is changed run :

```bash
npm run typegen

```

then to install dependencies  

```bash
npm install

``` 

then to start 

```bash
npm run dev 

``` 

To go to client  [http://localhost:3000](http://localhost:3000)  , 
To go to admin panal [http://localhost:3000/studio](http://localhost:3000/studio)
